Project Overview

StudentLoan The Student Loan System is a full-stack application aimed at streamlining the student loan process. It handles everything from loan applications and disbursement to repayment tracking and user management. This project was developed to demonstrate skills in database management, backend logic, and responsive front-end design.

Responsibilities: In this project, I'm responsible for the design and development of the Front-end and recording video presentations to explain the front-end code.

Key features

Online application submission system

Fill out the information and upload the supporting documents for the loan application all in one place.

Status tracking

Users can check the status of their applications in real-time.

Loan Borrower Verification System

The admin can check the borrower's status and can also view the form information in order to approve, return documents, or disapprove.
