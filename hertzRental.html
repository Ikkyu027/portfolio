Project Overview

Hertz Rental The Hertz Rental system is a full-stack application aimed at streamlining the car rental process. It handles everything from vehicle listings and availability to user authentication and booking management. This project was developed to demonstrate skills in database management, backend logic, and responsive front-end design.

Responsibilities: In this project, I was responsible for the UX/UI design work and creating the webpage (Front-end).

Key features

Search for smart cars

A system for searching and filtering cars by brand, model, type, and price range.

Online booking system

Users can check the date when the car is available and make a reservation immediately through the website.

Admin Dashboard

Admin page for managing car inventory, viewing booking reports, and customer information.
